WFSLive


WFSLive allows for real-time usage of the system, through the 8 analog inputs present in the MOTU soundcard connected to the laptop computer. 


Usage:


1 - Connect your sources to inputs 1 to 8 of the MOTU soundcard of the laptop.

2 - Go to menu entry View->Live.

3 - Adjust the global level (View->Level).

4 - Press the play button. 


Sources can be moved by draggint the points in the window.


Each track has level slider, an on/off button and a button to switch between point source and plane wave.


InputRouter



The InputRouter allows to select which input goes to which WFS track, and to apply EQ and compression to the input signal.


OSC control


Most parameters of each track together with many global parameters can be controlled via OSC, either from inside SuperCollider or from another program. The external program (i.e. max/msp) should send the osc commands to the language port of supercollider, which is usually 57120. To check what is the current port do NetAddr.langPort.


The OSC specification can be found here.


Examples:


b = NetAddr.new("127.0.0.1", 57120);

b = NetAddr.new("127.0.0.1", 57121);


//see the messages being sent

WFSLiveOSCResp.verbose = false

WFSLiveOSCResp.verbose = true



b.sendMsg('/wfscollider/source/position',0,-4,4)


b.sendMsg('/wfscollider/source/position_polar',0, -45,6)


b.sendMsg('/wfscollider/source/mute',0,false)


b.sendMsg('/wfscollider/source/level',0,0.1)


b.sendMsg('/wfscollider/source/type',0,\plane)


b.sendMsg('/wfscollider/source/set',0,\plane,-4,4,0.5)


b.sendMsg(*(['/wfscollider/all']++8.collect{

[[\point,\plane].choose,rrand(-5.0,5.0),rrand(-5.0,5.0),rrand(0.0,1.0 )]

}.flat

))


//global controls

b.sendMsg('/wfscollider/level',0.5)

b.sendMsg('/wfscollider/start')

b.sendMsg('/wfscollider/stop')

b.sendMsg('/wfscollider/eq/mid/q', 11)


//get info

b.sendMsg('/wfscollider/info/all')

b.sendMsg('/wfscollider/info/source',4)
